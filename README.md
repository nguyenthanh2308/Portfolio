## 🌸 Giới thiệu

Portfolio cá nhân được thiết kế theo phong cách **anime** — kết hợp giữa thẩm mỹ Nhật Bản hiện đại và trải nghiệm web mượt mà, tinh tế. Trang web không chỉ là nơi giới thiệu bản thân mà còn là một trải nghiệm trực quan độc đáo với nhạc nền, hiệu ứng hạt, và cursor tuỳ chỉnh.

## 📦 Cài đặt & Chạy local

> Không cần cài đặt bất kỳ package nào — đây là dự án thuần HTML/CSS/JS.

```bash
# Clone repository
git clone https://github.com/nguyenthanh2308/Portfolio.git

# Di chuyển vào thư mục
cd Portfolio

# Mở bằng Live Server (VS Code extension) hoặc mở thẳng file index.html
```

**Sử dụng VS Code:**
1. Cài extension [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)
2. Click chuột phải vào `index.html` → **Open with Live Server**

## 🌐 Deploy lên Vercel

Dự án được deploy tự động qua Vercel:

1. **Fork** hoặc **Clone** repository này
2. Đăng nhập [vercel.com](https://vercel.com) → **Add New Project**
3. Import từ GitHub → chọn repo này
4. Vercel tự nhận diện là **Static Site** — nhấn **Deploy**
5. Mỗi lần `git push` → Vercel tự động redeploy ⚡
