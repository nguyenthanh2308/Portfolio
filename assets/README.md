# 📁 Assets — Hướng dẫn sử dụng

Thư mục này chứa toàn bộ tài nguyên tĩnh cho portfolio.

## Cấu trúc

```
assets/
├── images/
│   ├── avatar.jpg          ← Ảnh đại diện của bạn (400×400px+)
│   ├── og-image.png        ← Ảnh preview khi share link (1200×630px)
│   └── projects/
│       ├── project-1.png   ← Thumbnail project 1 (800×500px)
│       ├── project-2.png   ← Thumbnail project 2
│       ├── project-3.png   ← Thumbnail project 3
│       ├── project-4.png   ← Thumbnail project 4
│       └── project-5.png   ← Thumbnail project 5
│
├── icons/
│   └── favicon.ico         ← Icon tab trình duyệt (32×32px)
│
└── music/
    ├── lofi/               ← Nhạc Lo-fi / Chill (định dạng .mp3)
    ├── synthwave/          ← Nhạc Synthwave / EDM (định dạng .mp3)
    └── ost/                ← Nhạc Anime OST (định dạng .mp3)
```

## Ghi chú
- Ảnh nên được nén để trang load nhanh (dùng https://squoosh.app)
- Nhạc nên ở định dạng .mp3 để tương thích tốt nhất
- Nếu chưa có ảnh, các placeholder emoji sẽ tự động hiển thị thay thế
